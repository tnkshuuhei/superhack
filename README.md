## Getting Started
setup frontend
```bash
npm install
# or
yarn install

```

```bash
cd smartcontracts

yarn install && yarn build
```
run frontend server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
setup .env file foth frontend/smartcontracts by following .envexample file

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
